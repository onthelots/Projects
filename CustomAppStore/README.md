# Custom-AppStore

<br>

## 앱 소개
📱 CustomAppStore : iTunes API를 활용한 커스텀 앱 스토어 프로젝트

<br>

## 구성
- 2개의 탭으로 구성 (앱 / 검색)
- Tab①(앱) : ScrollView(주제별 앱 리스트는 CollectionView / 가장 하단에 카테고리 별 리스트) 
- Tab②(검색) : SearchBar / 추천 앱 및 게임(인기순)
- 각각의 앱 정보(썸네일, 이름, 다운로드 외)는 별도의 DetailView로 만들기
